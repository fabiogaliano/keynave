KeyNav

Keyboard-driven UI navigation for macOS. Click anywhere on your screen without touching the mouse.

Download Source Code

Features

[A]

Hint Mode

Overlay alphabetic hints on all clickable elements. Type the hint to click - fast and precise navigation without the mouse.

[~]

Scroll Mode

Vim-style keyboard scrolling with hjkl keys. Multiple scrollable area selection and configurable speed.

[*]

Text Search

Type element names to search and auto-click. Find buttons by their text instead of memorizing hints.

[#]

Customizable

Full color customization with glass effects. Custom hotkeys, hint characters, and transparency levels.

How It Works

1

Activate

Press Cmd+Shift+Space to show hints over all clickable elements

2

Type

Type the hint characters (e.g., "AJ") to target an element

3

Click

Element is clicked automatically. Mode deactivates or refreshes for continuous clicking

Default Shortcuts

Activate Hint Mode Cmd + Shift + Space
Activate Scroll Mode Option + E
Cancel / Exit ESC
Right-click Ctrl + Enter
Clear search Option

Installation

Build from Source

git clone https://github.com/yourusername/KeyNav.git
cd KeyNav
xcodebuild -project KeyNav.xcodeproj -scheme KeyNav -configuration Release build

Requirements

  • macOS 14.0 or later
  • Xcode 15+ (for building)
  • Accessibility permissions (System Settings > Privacy & Security > Accessibility)
